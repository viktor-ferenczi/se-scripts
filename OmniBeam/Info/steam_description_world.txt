[h1]OmniBeam Arm Controller Test World[/h1]

[h2]Requirements[/h2]
[list]
[*] Experimental mode enabled in the game (so the Programmable Block works)
[*] Multigrid Projector client plugin via [url=https://github.com/sepluginloader/SpaceEngineersLauncher]Plugin Loader[/url]
[*] [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3218498375]OmniCorp: OmniTools[Lasers][/url] mod
[/list]

[h2]How to use the test world[/h2]
[list]
[*] Subscribe to the [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3216972580]OmniBeam Arm Control[/url] script
[*] Create and enter the world
[*] Join the Testers faction
[*] Spawn to the medical room
[*] Enter the pilot seat at the top
[*] Recompile the [b]Shipyard Program[/b] PB
[*] Welding controls on the pilot seat's toolbar: #3 Start, #4 Stop
[/list]

[h2]Remarks[/h2]
[list]
[*] Works both in creative and multiplayer
[/list]
